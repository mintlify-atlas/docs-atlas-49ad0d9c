## What is Modaic?

Modaic (Modular + Mosaic) is a Python framework that helps you compose and maintain production-ready DSPy applications. Built on top of DSPy, Modaic adds essential features for versioning, sharing, and reproducing your LLM programs.

## Why Modaic?

DSPy is powerful for building LLM programs, but production applications need more:

- **Version control**: Track changes to prompts and code as your program evolves
- **Sharing**: Collaborate with your team by sharing programs through the hub
- **Reproducibility**: Never lose your optimization progress - save and restore complete program states
- **Configuration management**: Cleanly separate program logic from configuration
- **Deployment**: Easy serialization and loading for production deployments

## How it works

Modaic extends DSPy with two main abstractions:

### PrecompiledProgram

Statically defined programs with explicit configuration:

```python
from modaic import PrecompiledProgram, PrecompiledConfig

class WeatherConfig(PrecompiledConfig):
    weather: str = "sunny"

class WeatherProgram(PrecompiledProgram):
    config: WeatherConfig

    def __init__(self, config: WeatherConfig, **kwargs):
        super().__init__(config, **kwargs)

    def forward(self, query: str) -> str:
        return f"The weather in {query} is {self.config.weather}."
```

### AutoProgram

Dynamically load programs from hub or local storage:

```python
from modaic import AutoProgram

program = AutoProgram.from_precompiled("username/weather-program", rev="v2.0.0")
result = program(query="Tokyo")
```
