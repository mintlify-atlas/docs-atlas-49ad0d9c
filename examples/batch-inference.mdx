Batch inference allows you to process large volumes of predictions efficiently and cost-effectively. Modaic supports batch processing across multiple providers including OpenAI, Anthropic, Together AI, and Fireworks AI.

## Overview

Batch inference is ideal for:
- Processing large datasets where immediate results aren't required
- Cost-effective inference (typically 50% cheaper than synchronous calls)
- High-volume question answering tasks
- Data labeling and classification at scale
- Offline evaluation and testing

<Info>
  Batch requests are processed asynchronously and can take from minutes to hours depending on the provider and queue size.
</Info>

## Prerequisites

Before starting, ensure you have:
- Modaic SDK installed: `pip install modaic`
- API keys for your chosen provider (OpenAI, Anthropic, Together AI, or Fireworks AI)
- Python 3.9 or higher

## Basic Usage

<Steps>
<Step title="Configure DSPy and create a predictor">

Set up your language model and create a simple predictor:

```python
import asyncio
import dspy
from modaic.batch import abatch
from modaic.batch.types import FailedPrediction

# Configure DSPy with your chosen provider
dspy.configure(
    lm=dspy.LM("openai/gpt-4o-mini"),
    adapter=dspy.ChatAdapter()
)

# Create a predictor with a signature
predictor = dspy.Predict("question -> answer")
```

</Step>

<Step title="Prepare your input data">

Create a list of input dictionaries matching your signature:

```python
inputs = [
    {"question": "What is the capital of France?"},
    {"question": "What is 2 + 2?"},
    {"question": "Who wrote Romeo and Juliet?"},
    {"question": "What is the largest planet?"},
    {"question": "Who painted the Mona Lisa?"},
    {"question": "What is the speed of light?"}
]
```

</Step>

<Step title="Submit batch request">

Use the `abatch()` function to process all inputs:

```python
async def main():
    print("Submitting batch request...")
    results = await abatch(predictor, inputs)
    
    # Process results
    for i, res in enumerate(results):
        pred = res.prediction
        if isinstance(pred, FailedPrediction):
            print(f"[{i}] FAILED: {pred.error}")
        else:
            print(f"[{i}] Q: {inputs[i]['question']}")
            print(f"    A: {pred.answer}")

if __name__ == "__main__":
    asyncio.run(main())
```

</Step>
</Steps>

## Advanced Configuration

You can customize batch processing behavior with additional parameters:

```python
results = await abatch(
    predictor,
    inputs,
    show_progress=True,      # Display progress bar
    poll_interval=10.0,      # Check status every 10 seconds
    max_poll_time="2h",      # Maximum wait time
    batch_size=100           # Custom batch size
)
```

## Error Handling

Always check for failed predictions in your results:

```python
from modaic.batch.types import FailedPrediction

successes = []
failures = []

for i, res in enumerate(results):
    pred = res.prediction
    if isinstance(pred, FailedPrediction):
        failures.append({
            "index": i,
            "input": inputs[i],
            "error": pred.error
        })
    else:
        successes.append({
            "index": i,
            "input": inputs[i],
            "output": pred
        })

print(f"Successes: {len(successes)}/{len(results)}")
print(f"Failures: {len(failures)}/{len(results)}")

# Retry failed predictions if needed
if failures:
    retry_inputs = [f["input"] for f in failures]
    retry_results = await abatch(predictor, retry_inputs)
```
